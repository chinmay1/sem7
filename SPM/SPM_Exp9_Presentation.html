CREATING A GIT REPO


~ $ git init blog 
Initialized empty Git repository in /home/pokemon/blog/.git/ 

~/blog $ ls -la
total 12K
drwxrwxr-x  3 pokemon pokemon 4.0K 2013-10-26 12:45 ./
drwxr-xr-x 49 pokemon pokemon 4.0K 2013-10-26 11:40 ../
drwxrwxr-x  8 pokemon pokemon 4.0K 2013-10-26 11:40 .git/
  • .git is directory tree all information of your repository
  • to un-git, simply delete .git folder

Setting UP your identity


git config --global user.name "Your Name"
git config --global user.email you@example.com

~/blog $ git config --global user.name "pikachu"
~/blog $ git config --global user.email "pikachu@pk.mn"~/blog $ git config --global -l
user.name=pikachu
user.email=pikachu@pk.mn
